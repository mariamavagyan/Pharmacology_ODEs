GOODNESS-OF-FIT METRICS
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

1. RÂ² (Coefficient of Determination)
   RÂ² = 1 - (SS_res / SS_tot)
   Where SS_res = Î£(y_obs - y_pred)Â²
         SS_tot = Î£(y_obs - mean(y_obs))Â²
   
   Range: 0 to 1 (higher is better)
   Interpretation: proportion of variance explained
   Pitfall: Can be artificially high with more parameters (use adjusted RÂ²)

2. RMSE (Root Mean Squared Error)
   RMSE = âˆš(1/n * Î£(y_obs - y_pred)Â²)
   
   Units: same as y (concentration, count, etc.)
   Use: compare fit quality across models with same data

3. AIC (Akaike Information Criterion)
   AIC = 2k + n*log(SSE/n)
   Where k = number of parameters, n = number of observations
   
   Use: Model comparison (lower AIC is better)
   Benefit: penalizes complexity

4. BIC (Bayesian Information Criterion)
   BIC = k*log(n) + n*log(SSE/n)
   
   Like AIC but stronger penalty for complexity
   Use: Model selection (lower is better)

DIAGNOSTIC PLOTS - WHAT THEY TELL YOU
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

1. OBSERVED vs PREDICTED
   â”œâ”€ Good: points scatter around diagonal y=x line
   â””â”€ Bad: systematic deviations (curved pattern, outliers)

2. RESIDUALS vs FITTED
   â”œâ”€ Good: random scatter around y=0 line
   â”œâ”€ Bad: curved pattern (model bias), funnel shape (heteroscedasticity)
   â””â”€ Action: may need model adjustment or transformation

3. HISTOGRAM of RESIDUALS
   â”œâ”€ Good: approximately normal distribution, centered at 0
   â”œâ”€ Bad: bimodal, skewed, heavy tails
   â””â”€ Implies: measurement error not normally distributed

   â”œâ”€ Good: points follow diagonal line
   â”œâ”€ Bad: S-shape or departure at extremes
   â””â”€ Tests: normality of residuals

5. SCALE-LOCATION PLOT
   â”œâ”€ Good: random scatter, no trend
   â”œâ”€ Bad: funnel or curved pattern
   â””â”€ Tests: homoscedasticity (constant variance)

6. RESIDUALS vs ORDER
   â”œâ”€ Good: random scatter
   â”œâ”€ Bad: cyclic pattern, increasing/decreasing trend
   â””â”€ Tests: independence of measurements

â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
Advanced validation for population data:

1. Simulate many subjects from fitted model
2. For each observation time, compute percentiles (5th, 50th, 95th)
3. Compare simulated percentiles to observed percentiles
4. Observed percentiles should fall within simulated range

Success: observed data fall within prediction interval
Failure: systematic bias (try different model structure)

PARAMETER IDENTIFIABILITY
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Check with:
1. Correlation matrix (high correlations = identification problem)
2. Parameter confidence intervals (infinite = unidentifiable)
3. Profile likelihood plots (flat = poorly identified)

Solutions:
- Fix poorly identified parameters to literature values
- Reduce model complexity
- Collect more/better data (different sampling times)
